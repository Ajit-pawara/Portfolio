# Ajit Pawara | Cybersecurity Portfolio

A terminal-inspired portfolio showcasing my 90-day cybersecurity learning challenge, skills, and projects.

🌐 **Live Website**: [https://ajit-pawara.github.io/Portfolio/](https://ajit-pawara.github.io/Portfolio/)

## Features

- **90-Day Challenge Tracker**: Interactive progress grid with log viewer and HTML live previews.
- **Skills Radar**: Custom, responsive SVG-based skill radar chart and pathway matrices.
- **Project Dossier**: Offensive and defensive project showcase with an interactive code viewer.
- **Admin Control Panel**: Privileged settings panel (authenticated as `root`) to customize profile and track logs locally.

## Tech Stack

- **Framework**: React 19 (TypeScript)
- **Bundler**: Vite
- **Styling**: Modern custom CSS with tailwindcss modules
- **Icons**: Lucide React
- **Build & Quality**: oxlint (linter), tsc (TypeScript compiler)

## Local Development

To run the project locally, install dependencies and start the development server:

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Run linting
npm run lint

# Build for production
npm run build
```
