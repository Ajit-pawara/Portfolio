# Ajit Pawara | Cybersecurity Portfolio

A terminal-inspired portfolio showcasing my 90-day cybersecurity learning challenge, skills, and projects.

🌐 **Live**: https://ajit-pawara.github.io/Portfolio/

## Features

- **90-Day Challenge Tracker**: Interactive progress grid
- **Skills Radar**: Visual skill distribution chart
- **Project Dossier**: Offensive and defensive project showcase
- **Admin Panel**: Edit mode to customize portfolio data

## Tech Stack

- HTML5, CSS3, JavaScript
- Chart.js for radar visualization
- Lucide icons
- Responsive design

## Usage

Simply open `index.html` in your browser, or visit the GitHub Pages link above.
